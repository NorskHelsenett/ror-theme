# PrimeNG Theming with SASS

# Compile

## Prerequisites

sass, install with

`npm install -g sass`

## Create a css

Dark:

`sass --update themes/rortheme/dark/teal/theme.scss:themes/rortheme/result/theme-dark.css`

Light:

`sass --update themes/rortheme/light/teal/theme.scss:themes/rortheme/result/theme-light.css`
